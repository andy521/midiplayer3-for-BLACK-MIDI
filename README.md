midiplayer by ax_pokl. 
=============
* Drag file to play. 
* Support command line. 
* Support mid and rmi file. 

Mouse Control
-------------
* MLeft	Seek/Keyboard
* MRight	Pause
* MWheel	Volume

Key Control
-------------
* L/R	Seek
* UP/DN	Volumn
* +/-	Speed
* Space	Pause

File Control
-------------
* PGUP	Next
* PGDN	Prev
* Home	First
* End	Last

Other Functions
-------------
* F1	Help
* F2	Software Reset
* F3	Hardware Reset
* F5/F6	FPS
* F7/F8	Bar Size
* F9	Bar Color
* F11	Key Number
* F12	Loop Mode
